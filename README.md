# **Betasharp Launch Script**
Backup\Launch tool for [BetaSharp](https://github.com/betasharp-official/betasharp) ( Minecraft Beta 1.7.3 written in C# )


Nothing much, it's just QoL and it's not necessary since pretty much all it does is run `cd BetaSharp.Launcher && dotnet run --configuration Release` with a few extra steps

## #What does it do?
  * ✨Double click to start ✨
  * Backup saves
  * Start the Launcher
###### almost as if it was a minor QoL

## #Where are my Backups?
  By default it should be in `%appdata%\.BetaSharp\savesBackup` you can paste that in `Win + R`<br/>
  Also by default inside that folder should be the .zips of the whole `save` folder using the following naming scheme `saves-dd-MM-yyyy_HH-mm-ss.zip` for example `saves-31-10-2025_12-59-31.zip`
  
  > dd=Day<br/>
  MM=Month<br/>
  yyyy=Year<br/>
  HH=Hour<br/>
  mm=Minutes<br/>
  ss=Seconds<br/>
